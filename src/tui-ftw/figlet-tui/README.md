Figlet TUI

- Terminal-style demo (keyboard-first)
- Commands supported:
  - `figlet [-s|--stream] <text>` — render ASCII (streaming option)
  - `font <name>` — change font
  - `size <n>` — set preview size
  - `list-fonts [query]` — show available fonts
  - `clear`, `help`
- Auto-preloads all FIGlet fonts in the background so no runtime downloads are required.
